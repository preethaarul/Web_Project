QuizHub – Online Quiz Platform

QuizHub is a full-stack web application that allows users to create, take, and share quizzes easily.
This project demonstrates a clean frontend-backend structure with a simple user authentication flow.

TECH STACK USED:
Frontend -  - HTML5, CSS3, JavaScript
Backend - Node.js, Express.js
Database: In-memory storage (resets when server restarts)
(Can be extended to real databases like MongoDB/MySQL)

FEATURES:
User registration and login
Create custom quizzes with multiple questions
Take quizzes and view results instantly
Share created results
Dashboard showing created, taken quizzes  and average score.
Responsive UI with plain HTML and CSS
